Primer Nivel

Sensor de pulso cardíaco en Arduino

Propuesta

Desarrollo de una muñequera inteligente vinculada a una aplicación móvil web, diseñada no solo para monitorear el pulso cardíaco del usuario, sino también para detectar anomalías en tiempo real. Cuando el brazalete identifique un ritmo cardíaco irregular o peligroso, enviará una notificación inmediata a través de la aplicación, alertando al usuario o al cuidador.

Pero esto no es todo. Este dispositivo ha sido concebido especialmente para brindar apoyo a pacientes en situación de vulnerabilidad o con enfermedades terminales, tales como personas de la tercera edad, pacientes con antecedentes de enfermedades cerebrales, parálisis parcial o total, entre otros.

Quinto Nivel


Desarrollo de un brazo robótico con Machine Learning y Reconocimiento Visual para el Control y juego de 3 en Raya

Propuesta

La capacidad de un sistema robótico para percibir y comprender un entorno visual, como el estado de un juego de mesa, abre nuevas avenidas para la aplicación del aprendizaje automático. Un ejemplo claro es la implementación de un brazo robótico que no solo se controle mediante joysticks, sino que también pueda “ver” y reaccionar al tablero de 3 en raya. Esto implica desde algoritmos de procesamiento de imágenes e interpretación de los objetos y el diseño de algoritmos que permitan al robot interpretar el estado del juego.

Algunos objetivos del sistema:

  • Diseñar e implementar un sistema de control basado en Arduino para el movimiento de las articulaciones del brazo robótico, utilizando servomotores y sensores para lograr precisión y fluidez en sus movimientos.
  • Programar la interfaz en Arduino IDE que permita realizar tareas específicas, como levantar, agarrar y mover objetos, integrando módulos de entrada como joysticks, botones o controladores inalámbricos.
  • Desarrollar algoritmos de Machine Learning para interpretar el estado del tablero de 3 en raya mediante visión por computadora desde las fichas y determinando cuándo se ha producido una jugada válida.
  • Integrar el reconocimiento visual con el control del sistema, de modo que el brazo robótico interprete el juego real y reaccione al estado del juego, colocando una ficha en un casillero, o respondiendo a la jugada del oponente.
  • Explorar la posibilidad de utilizar aprendizaje reforzado para que el brazo robótico aprenda a jugar al 3 en raya de manera autónoma.
Nota: Un brazo robótico controlado mediante joysticks y desarrollado sobre la plataforma Arduino.
